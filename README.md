# Omix Solutions - Digital Agency Website

A modern, animated website for Omix Solutions with dark theme, neon glows, and 3D animations inspired by Blitzit.app.

## Features

- 🎨 Dark theme with neon animated glows
- ✨ 3D animations and particle effects
- 🌈 Animated border glows
- 📱 Fully responsive design
- ⚡ Built with Next.js 14 and React
- 🎭 Framer Motion animations
- 🎯 Modern UI/UX

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Three.js** - 3D effects (via React Three Fiber)
- **Lucide React** - Icons

## Customization

- Edit colors in `tailwind.config.js` under the `neon` theme
- Modify content in component files
- Adjust animations in component files using Framer Motion
